Page Manipulation

setTitle(title)

Set the title of the web page.

setHeader(html)

Set the footer to the string of HTML.

appendHeader(html)

Add a string of HTML to the end of the footer.

prependHeader(html)

Add a string of HTML to the beginning of the footer.

setFooter(html)

Set the footer to the string of HTML.

appendFooter(html)

Add a string of HTML to the end of the footer.

prependFooter(html)

Add a string of HTML to the beginning of the footer.

loadExternalCSS(href)

Load a CSS into the page from the given URL.

loadExternalJS(href)

Load a JavaScript script into the page from the given URL.

Help

So, you want to try use this thing? I'd recommend loading the defaults.
Anyway, here is what the view page contains:
  1. JavaScript from Global, Scene, and HotSpot entries
  2. JavaScript and CSS inserted from loadExternalJS() and loadExternalCSS() used above
  3. CSS from Global, Scene, and HotSpot entries
  4. Header HTML
  5. Scene container
  6. Footer HTML
Note that event scripts (e.g. click, enter, hover) are bound to the respective callback functions.
Check out the API documentation for things to put in JS fields. And remember, you can declare functions and variables anywhere you like in the general JS entries.
